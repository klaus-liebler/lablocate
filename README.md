## Components 
- Quovo DWM3000 ist ein UWB-Ortungsmodul (UltaWideBand_ige Funkwellen lassen sich gut hinsichtlich Signallaufzeit vermessen)
- ESP32-C6: Mikrocontroller mit 2.4 GHz Wi-Fi 6 (802.11ax), Bluetooth® 5 (LE), Zigbee and Thread (802.15.4)
- LSM6DS3 ist ein 6DOF-Beschleunigungssensor
- TPS63802 ist ein StepUp-Step-Down-Converter
- TP4057 ist ein Batterie-Lade-ID
